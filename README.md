## 📚 這份教材有什麼？

共 **27 題**，從基礎到進階，每題都有四個步驟：

| 步驟 | 內容 |
|------|------|
| 📖 概念 | 用白話文解釋這題在學什麼 |
| 🔍 追蹤 | 直接執行範例，看程式怎麼一步一步跑 |
| ✏️ 填空 | 把 `___` 填入正確答案（不會可以點 `?` 看提示）|
| 💡 挑戰 | 自己從空白開始寫，完成後按「✓ 完成」記錄進度 |

## 💡 使用小提醒

- 建議用 **Chrome 或 Edge** 開啟效果最好
- 需要網路連線才能使用（Python 環境從網路載入）
